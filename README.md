# Udacity-iOS
Collection of projects for the Udacity iOS Nanodegree program

## PitchPerfect

* Allows user to record their voice and play back the recording in multiple modes

## MemeMe v1

* User can take a picture or select an image from their phone, and annotate it with text

## MemeMe v2

* Expands on MemeMe v1 functionality with the ability to create multiple memes, display them in table view, and share them

## On The Map

* User can sign into the Udacity website and share their location and a link about themselves with other students
